ACRE is a civilization-scale systems framework focused on
reducing irreversible error under conditions of high complexity.

## What ACRE Does
- supports early risk detection
- preserves reversibility in governance
- enables recursive self-correction

## What ACRE Does Not Do
- make decisions
- define values
- predict futures
- replace political judgment

## Design Principles
- no centralization of power
- no irreversible execution
- no automated value decisions
- full auditability
- expiration by default

This repository contains documentation and specifications only.
It is not an implementation mandate.
